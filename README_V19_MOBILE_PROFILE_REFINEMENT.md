# Gemstone V19 — Mobile Profile Refinement

This version improves the cellphone view of the Profile subpages.

## Improvements

- Withdrawal form uses tighter spacing and compact wallet summary chips.
- Referral statistics display two cards per row.
- Wallet Activity is converted from a wide table into stacked mobile cards.
- Cash-in history is converted into stacked cards.
- Long dates, transaction types, amounts, and reference numbers wrap correctly.
- Referral code and copy controls fit narrow screens.
- No horizontal table clipping.

## Installation

No new Supabase SQL is required.

1. Upload the V19 files to GitHub Pages.
2. Keep your current `config.js`.
3. Refresh with Ctrl + Shift + R.
4. Close and reopen the APK or browser tab to clear cached CSS.

The APK does not need to be rebuilt because it loads the live website.
