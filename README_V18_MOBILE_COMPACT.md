# Gemstone V18 — Mobile Compact Layout

This version keeps every V17 feature and optimizes the website for cellphone screens.

## Home page

- Wallet balance, Available points, and Active gemstones display in one row.
- Purchased gemstones display three cards per row.

## Membership page

- Membership products display three cards per row.

## Profile page

- Wallet balance and Total memberships display in one row.
- Add Funds, Withdrawal, Referral, and Wallet Activity display in one row.
- Action subtitles are hidden on small screens to preserve space.

## Installation

No new Supabase SQL is required.

1. Upload the V18 website files to GitHub Pages.
2. Keep your current `config.js`.
3. Refresh the website using Ctrl + Shift + R.
4. On Android, close and reopen the APK if the old CSS is still cached.

The APK loads the live GitHub Pages website, so no new APK build is necessary.
