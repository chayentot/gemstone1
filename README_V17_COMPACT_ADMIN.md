# Gemstone V17 — Compact Admin Command Sections

This version keeps all V16 features and adds four command buttons to minimize the admin page:

- Pending Reviews
- Wallet Withdrawals
- Gemstone Management
- Registered Users

Only one section is displayed at a time. The last opened section is remembered on the device.

## Installation

No new SQL is required when V16 is already installed.

1. Upload the updated website files to GitHub Pages.
2. Keep your existing `config.js`.
3. Refresh the admin page with Ctrl + Shift + R.

When V16 has not yet been installed, run `v16_admin_gemstone_management.sql` first.

The APK will show this layout automatically because it loads the live GitHub Pages website.
