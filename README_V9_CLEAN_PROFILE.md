# Gemstone V9 — Clean Profile Action Buttons

This update reorganizes the Profile page into five large icon buttons:

- Add Funds
- Withdrawal
- Referral
- Wallet Activity
- Point Activity

Only the selected section is displayed, making the page cleaner on desktop and mobile.

## Install

No SQL changes are required if V8 is already installed.

Upload the updated files to GitHub, especially:

- profile.html
- profile.js
- style.css

You may upload the entire package and replace existing files.

After GitHub Pages redeploys, refresh with Ctrl + Shift + R.
