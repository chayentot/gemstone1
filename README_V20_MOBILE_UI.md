# Gemstone V20 — Mobile UI

This version keeps all V19 features and further improves the cellphone layout.

## Referral section

- Total referrals, Active referrals, Total referred purchases, and Total referral rewards now display in one row.
- Labels and values resize automatically on narrow screens.
- Referral network entries display as compact mobile cards.
- Empty referral messages span the full width.

## Transactions

- Wallet activity, cash-in history, and withdrawal history use compact stacked cards on mobile.
- Long transaction names, amounts, dates, and reference numbers wrap instead of being cut off.

## Installation

No new Supabase SQL is required.

1. Upload the V20 files to GitHub Pages.
2. Keep your current `config.js`.
3. Refresh the site using Ctrl + Shift + R.
4. Close and reopen the APK or mobile browser to clear cached CSS.

No APK rebuild is required because the app loads the live website.
