# Gemstone V23 — Final Mobile Polish

V23 keeps all V22 features and adds final mobile refinements.

## Improvements

- Shorter top header and bottom navigation
- Stronger active-page highlight
- Bottom navigation hides while the phone keyboard is open
- Better numeric input behavior for money and phone fields
- Global loading overlay and spinner support
- Buttons can be safely disabled while actions are processing
- Global success and error toast support
- Improved form and dialog spacing
- Cache-busted CSS and JavaScript files

## Installation

No new Supabase SQL is required.

1. Upload the V23 files to GitHub Pages.
2. Preserve your current `config.js`.
3. Force-refresh the browser.
4. Fully close and reopen the APK.

The APK does not need to be rebuilt because it loads the live website.
