# Gemstone website — images in repository root

Upload every file in this folder directly to the root of your GitHub repository.

The gemstone image files must appear beside `index.html`, for example:

- quartz.webp
- amethyst.webp
- topaz.webp
- garnet.webp
- aquamarine.webp
- opal.webp
- sapphire.webp
- emerald.webp
- ruby.webp
- diamond.webp

This version does not use an `images` folder.
